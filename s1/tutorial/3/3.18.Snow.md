# Snowcone、AWS SAP认证与Snowmobile现状

## 1. Snowcone 状态

**是的，Snowcone已完全停止使用**。

**时间线**：
- **2024年11月12日**：AWS正式停售所有Snowcone设备（HDD和SDD两种型号），**新老客户均无法再下单**
- **2025年11月12日**：AWS完全终止对所有Snowcone设备的技术支持，**所有Snowcone设备彻底无法使用**

**现状**：目前(2025年12月)Snowcone已进入"**完全终止阶段**"，无任何官方支持渠道，设备无法正常工作。

## 2. AWS SAP认证考试状态

**AWS SAP认证考试（SAP-C02和PAS-C01）仍然正常进行**，**不受Snowcone停服影响**。

**关键说明**：
- 虽然Snowcone已停止服务，但**AWS认证考试内容会定期更新**，移除已停止服务的技术细节
- 最新版考试大纲中，Snowcone相关内容已被替换为当前主流的**Snowball Edge**和**DataSync**等服务
- 考试重点转向SAP在AWS上的架构设计、部署优化和管理，而非特定硬件设备操作

## 3. Snowmobile现状

**Snowmobile已在2024年3月完全停止服务**。

**终止细节**：
- 2024年3月，AWS**从官网彻底移除**Snowmobile服务页面，所有相关链接重定向至Snowball Edge等替代方案
- 官方确认**不再提供**任何Snowmobile设备租赁或使用服务，无后续替代产品
- 当前状态：**完全退役，无技术支持，无法使用**

## 总结

| 服务 | 当前状态 | 停止时间 | 替代方案 |
|------|---------|---------|---------|
| Snowcone | **完全停止** | 2025年11月12日 | Snowball Edge、DataSync |
| AWS SAP认证 | **正常运行** | 未宣布 | 无(考试内容已更新) |
| Snowmobile | **完全停止** | 2024年3月 | Snowball Edge(大型数据迁移) |

**建议**：如果您是Snowcone或Snowmobile的现有客户，强烈建议尽快迁移至Snowball Edge或AWS DataSync等替代服务，避免服务中断影响业务。
