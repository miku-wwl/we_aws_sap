#### 📝 [391/529] AWS IoT Core 物联网核心服务

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 需要连接和管理大量 IoT 设备。
* **需求：** 设备认证，消息路由。
* **约束：** 大规模设备连接，可靠通信。
* **IoT 目标：** 构建可扩展的 IoT 应用。

**2. ⚡ 秒杀思路**
* **IoT 连接方案：**
    * **自定义 MQTT 服务器:** 开发维护成本高。
    * **第三方平台:** 集成复杂，成本高。
    * **IoT Core:** AWS 原生 IoT 平台。

**3. ✅ 正确选项解析**
* **IoT Core:** 企业级 IoT 平台，提供设备连接、认证和消息路由。

**5. 📚 核心考点:** IoT Core 在物联网应用开发中的配置和最佳实践。
