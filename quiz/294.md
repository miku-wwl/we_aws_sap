#### 📝 [294/529] SageMaker Canvas 无代码机器学习

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 业务分析师需要构建预测模型。
* **技能要求：** 无编程背景，需要可视化界面。
* **数据源：** S3 中的结构化数据。
* **目标：** 快速原型开发，自动特征工程。

**2. ⚡ 秒杀思路**
* **机器学习平台选择：**
    * **SageMaker Studio:** 需要 Python 技能。
    * **SageMaker Autopilot:** 主要通过 API 使用。
    * **QuickSight ML:** 仅限预测分析。
    * **SageMaker Canvas:** 可视化无代码平台。
* **Canvas 核心功能：**
    * **可视化建模:** 拖拽式模型构建。
    * **自动 ML:** 自动特征工程和模型选择。
    * **模型解释:** 可解释性报告。
    * **一键部署:** 快速部署为端点。
* **实施策略：**
    * **选项 A (Studio Notebook):** 需要编程技能。
    * **选项 B (Autopilot API):** 技术门槛高。
    * **选项 C (Canvas):** 
        * 可视化界面。
        * 无需编程经验。
        * 快速原型开发。
    * **选项 D (Custom ML):** 开发周期长。
* **使用流程：**
    * 导入数据到 Canvas。
    * 选择目标列和问题类型。
    * Canvas 自动生成和评估模型。
    * 部署最佳模型为预测端点。

**3. ✅ 正确选项解析 (选项 C)**
* **SageMaker Canvas:** 业务友好的无代码机器学习平台，降低 AI 应用门槛。

**5. 📚 核心考点:** Canvas 在企业 ML 应用普及中的作用和使用场景。
