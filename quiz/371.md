#### 📝 [371/529] AWS Certificate Manager SSL/TLS 证书管理

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 需要为多个服务管理 SSL/TLS 证书。
* **需求：** 自动证书部署，续期管理。
* **约束：** 多种 AWS 服务集成，免费证书。
* **安全要求：** 自动证书更新，安全传输。

**2. ⚡ 秒杀思路**
* **证书管理挑战：**
    * **手动购买:** 成本高，管理复杂。
    * **自签名证书:** 浏览器不信任，安全性差。
    * **ACM:** 免费 SSL/TLS 证书管理。
* **ACM 核心功能：**
    * **免费证书:** 完全免费的 SSL/TLS 证书。
    * **自动部署:** 自动部署到集成服务。
    * **自动续期:** 自动证书续期管理。
    * **多服务集成:** ELB、CloudFront、API Gateway 等。
* **管理优势：**
    * **选项 A (第三方):** 成本高，管理复杂。
    * **选项 B (手动):** 续期困难，容易出错。
    * **选项 C (ACM):** 
        * 免费证书服务。
        * 自动部署和续期。
        * 无缝 AWS 集成。
    * **选项 D (无证书):** 安全风险高。
* **使用场景：**
    * Web 应用 HTTPS。
    * API 安全传输。
    * CloudFront 分发。
    * 负载均衡器安全。

**3. ✅ 正确选项解析 (选项 C)**
* **ACM:** 免费 SSL/TLS 证书管理服务，简化证书部署和续期。

**5. 📚 核心考点:** ACM 在网络安全中的应用和配置方法。
