#### 📝 [385/529] AWS Well-Architected Framework 架构框架

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 需要评估和改进架构质量。
* **需求：** 架构标准，最佳实践。
* **约束：** 复杂系统架构，多维度评估。
* **目标：** 构建云原生最佳架构。

**2. ⚡ 秒杀思路**
* **架构评估方案：**
    * **经验评估:** 主观性强，标准不一。
    * **自定义框架:** 开发成本高。
    * **Well-Architected:** AWS 官方架构框架。

**3. ✅ 正确选项解析**
* **Well-Architected:** AWS 官方架构框架，提供云原生最佳实践指导。

**5. 📚 核心考点:** Well-Architected 六大支柱在企业架构设计中的应用。
