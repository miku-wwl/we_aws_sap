#### 📝 [389/529] AWS AppSync GraphQL 服务

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 需要构建 GraphQL API。
* **需求：** 实时数据同步，离线支持。
* **约束：** 复杂的数据查询，多数据源。
* **开发目标：** 简化 API 开发和管理。

**2. ⚡ 秒杀思路**
* **API 开发方案：**
    * **REST API:** 需要多次请求，效率低。
    * **自定义 GraphQL:** 开发复杂。
    * **AppSync:** 托管式 GraphQL 服务。

**3. ✅ 正确选项解析**
* **AppSync:** 托管式 GraphQL 服务，简化数据 API 开发和实时同步。

**5. 📚 核心考点:** AppSync 在现代 API 开发中的优势和配置方法。
