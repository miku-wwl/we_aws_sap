#### 📝 [388/529] AWS Data Pipeline 数据管道

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 需要在不同服务间移动和转换数据。
* **需求：** 可靠数据传输，错误处理。
* **约束：** 复杂的数据依赖关系。
* **处理目标：** 自动化数据工作流。

**2. ⚡ 秒杀思路**
* **数据传输方案：**
    * **自定义脚本:** 维护困难，可靠性低。
    * **第三方工具:** 集成成本高。
    * **Data Pipeline:** 托管式数据工作流。

**3. ✅ 正确选项解析**
* **Data Pipeline:** 托管式数据工作流服务，可靠移动和转换数据。

**5. 📚 核心考点:** Data Pipeline 在数据工程中的应用和配置。
