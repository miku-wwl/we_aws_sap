#### 📝 [349/529] AWS License Manager 许可证管理

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** 需要管理软件许可证的合规性和成本。
* **需求：** 集中许可证管理，避免违规使用。
* **约束：** 支持多种许可证类型，跨账户跟踪。
* **合规要求：** 满足软件供应商的许可证条款。

**2. ⚡ 秒杀思路**
* **许可证管理挑战：**
    * **手动跟踪:** 容易出错，难以规模化。
    * **分散管理:** 缺乏全局视图。
    * **合规风险:** 可能违反许可证条款。
* **License Manager 解决方案：**
    * **集中管理:** 统一管理所有软件许可证。
    * **合规检查:** 自动检查许可证使用合规性。
    * **跨服务集成:** 与 EC2、RDS 等服务集成。
    * **成本优化:** 避免不必要的许可证购买。
* **管理优势：**
    * **选项 A (Excel):** 手动维护，容易出错。
    * **选项 B (自定义):** 开发成本高，功能有限。
    * **选项 C (License Manager):** 
        * 企业级许可证管理。
        * 自动合规检查。
        * 多供应商支持。
    * **选项 D (无管理):** 合规风险高，成本浪费。
* **配置要点：**
    * 导入许可证信息。
    * 定义许可证规则。
    * 配置合规检查。
    * 监控使用情况和成本。

**3. ✅ 正确选项解析 (选项 C)**
* **AWS License Manager:** 企业级软件许可证管理服务，确保合规性和成本优化。

**5. 📚 核心考点:** License Manager 在软件资产管理中的应用和配置方法。
