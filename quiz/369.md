#### 📝 [369/529] AWS Network Firewall 高级防火墙

**1. 🕵️‍♂️ 题眼与约束分析**
* **场景：** VPC 级别的网络流量过滤。
* **需求：** 细粒度网络控制，应用层过滤。
* **约束：** 多 VPC 统一管理，复杂规则配置。
* **安全要求：** 深度包检测，威胁情报集成。

**2. ⚡ 秒杀思路**
* **网络防火墙方案：**
    * **NACL:** 简单的包过滤，功能有限。
    * **Security Group:** 状态控制，无深度检测。
    * **Network Firewall:** 企业级网络防火墙。
* **Network Firewall 特性：**
    * **深度包检测:** 应用层流量分析。
    * **规则引擎:** 强大的规则配置能力。
    * **威胁情报:** 自动威胁情报集成。
    * **集中管理:** 多 VPC 统一管理。
* **防火墙优势：**
    * **选项 A (NACL):** 功能简单，无法应用层过滤。
    * **选项 B (传统防火墙):** 部署复杂，成本高。
    * **选项 C (Network Firewall):** 
        * 企业级防火墙功能。
        * VPC 原生集成。
        * 高级威胁检测。
    * **选项 D (无防火墙):** 安全风险高。
* **配置要点：**
    * 创建防火墙策略和规则组。
    * 配置 VPC 子网关联。
    * 设置威胁检测和日志记录。
    * 监控防火墙性能和告警。

**3. ✅ 正确选项解析 (选项 C)**
* **Network Firewall:** 企业级网络防火墙，提供 VPC 级别的高级安全防护。

**5. 📚 核心考点:** Network Firewall 在云网络安全中的配置和最佳实践。
