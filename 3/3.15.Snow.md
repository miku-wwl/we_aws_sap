# 是的！2025年11月7日后，所有Snowball Edge设备将**只向现有客户提供新订单服务**，**新客户将无法申请**。

## 这句话的准确含义

**现有客户**：已在AWS注册并使用过Snowball Edge服务的用户

**新客户**：从未使用过Snowball Edge服务的AWS用户

**服务限制**：
- 2025年11月7日起，**新客户将无法创建任何Snowball Edge服务订单**（包括设备租赁和使用）
- **现有客户仍可继续使用已有的Snowball Edge设备**，不受影响
- 现有客户在2025年11月7日后**仍可申请新的Snowball Edge订单**（设备租赁、使用等）

## 官方说明

AWS官方文档明确表示：

> "**Effective November 7, 2025, AWS Snowball Edge will only be available to existing customers.** If you would like to use AWS Snowball Edge, sign up prior to that date. New customers should explore AWS DataSync for online transfers, AWS Data Transfer Terminal for secure physical transfers, or AWS Partner solutions. For edge computing, explore AWS Outposts."

翻译："自2025年11月7日起，AWS Snowball Edge将只向现有客户提供服务。如果您想使用AWS Snowball Edge，请在该日期前注册。新客户应考虑使用AWS DataSync进行在线传输，AWS Data Transfer Terminal进行安全物理传输，或AWS合作伙伴解决方案。对于边缘计算，可考虑AWS Outposts。"

## 为什么AWS这样做？

这是AWS产品线调整的一部分，多个官方文档显示，Snowball Edge正被逐步整合到其他服务中：

- 对于**数据迁移**：新客户被引导至AWS DataSync（在线）或AWS Data Transfer Terminal（物理设备）
- 对于**边缘计算**：新客户被引导至AWS Outposts服务

## 对用户的影响

**1. 对现有客户**：
- 无影响，可继续使用并申请新订单
- 但应考虑长期替代方案，因为这通常是服务生命周期的一个阶段

**2. 对新客户**：
- 2025年11月7日后**完全无法使用Snowball Edge服务**
- 必须选择替代方案：
  - 数据迁移：AWS DataSync（在线）或AWS Data Transfer Terminal（物理）
  - 边缘计算：AWS Outposts
  - 或考虑其他AWS合作伙伴解决方案

## 总结

这句话意味着Snowball Edge服务正进入"维护阶段"，不再向新客户开放，但现有客户仍可继续使用。这是AWS服务更新迭代的常见做法，目的是引导用户使用更新、更整合的解决方案。
